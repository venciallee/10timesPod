# 10timesPod HTML Demo

这是 10timesPod 的交互式 HTML 原型，展示了产品的核心功能和用户界面。

## 📁 文件说明

- `index.html` - 完整的单页应用原型

## 🎯 展示的功能

### 1. 发现页面
- 热门播客推荐网格
- 搜索栏与 RSS 订阅入口
- 最新更新的单集列表
- 一键订阅按钮

### 2. 订阅管理页面
- 已订阅播客的网格展示
- 未听集数标记 (badge)
- 快速播放最新集按钮
- 网格/列表视图切换

### 3. 播放页面（核心功能）
- **左侧**: 播客信息与播放控件
  - 播客封面
  - 播放/暂停、快进/快退
  - 进度条
  - 播放速度控制 (0.75x - 1.5x)
  - 字幕显示模式切换

- **右侧**: 双语字幕实时展示
  - 英文原文 + 中文翻译逐句对照
  - 当前播放句子自动高亮
  - 点击任意句子跳转到对应时间
  - 时间戳显示
  - 三种显示模式：仅英文、仅中文、双语对照

### 4. 设置页面
- **AI 模型配置**
  - 服务提供方选择 (OpenRouter / OpenAI / 自定义)
  - API Key 输入（密文显示）
  - Base URL 配置（自定义端点）
  - 转录模型选择
  - 翻译模型选择
  - 连接测试功能

- **播放设置**
  - 自动续播开关
  - 字幕自动滚动开关
  - 默认播放速度设置

- **账户设置**
  - 邮箱显示
  - 退出登录

### 5. 底部常驻播放栏
- 当前播放的播客信息
- 播放控制按钮
- 进度条
- 时间显示
- 全局可用，切换页面不中断播放

## 🎨 技术特性

- **响应式设计**: 使用 Tailwind CSS 构建，适配桌面和平板
- **渐变背景**: 深色主题，紫色渐变背景
- **交互动效**: hover 效果、过渡动画
- **状态管理**: JavaScript 控制页面切换和显示模式
- **组件化设计**: 卡片、按钮、输入框等使用统一风格

## 🚀 如何使用

### 直接在浏览器中打开

```bash
# 方式 1: 直接用浏览器打开文件
open demo/index.html

# 方式 2: 使用 Python 启动本地服务器
cd demo
python3 -m http.server 8000
# 然后访问 http://localhost:8000

# 方式 3: 使用 Node.js 的 http-server
npx http-server demo -p 8000
```

### 交互说明

1. **页面导航**: 点击顶部导航栏的"发现"、"订阅"、"设置"切换页面
2. **进入播放页**: 点击任意播客卡片或"播放"按钮进入播放页
3. **字幕交互**: 
   - 点击字幕句子跳转到对应时间
   - 使用顶部按钮切换显示模式（仅英文/仅中文/双语）
4. **播放控制**: 点击播放按钮（图标会切换）
5. **返回**: 在播放页点击左上角"← 返回"按钮回到发现页

## 📐 设计规范

### 颜色方案
- **主色**: 蓝色 (#3B82F6)
- **背景**: 深灰渐变 + 紫色点缀
- **文字**: 白色主文字，灰色辅助文字
- **卡片**: 半透明深灰背景 + 边框

### 字体大小
- **标题**: text-3xl (30px)
- **副标题**: text-xl (20px)
- **正文**: text-base (16px)
- **辅助文字**: text-sm (14px)
- **微型文字**: text-xs (12px)

## 🔄 与 PRD 的对应关系

此 HTML 原型实现了 PRD 中定义的以下核心用户流程：

1. ✅ F1: 播客订阅与管理
2. ✅ F2: 播客播放器
3. ✅ F3: 英文转录与字幕同步
4. ✅ F4: 中文翻译与双语对照
5. ✅ F5: 模型设置页
6. ⚠️ F6: 用户系统（仅 UI，无实际认证）

## 📝 注意事项

- 这是一个**静态原型**，不包含后端功能
- 音频播放、转录、翻译等功能为模拟展示
- API Key 输入不会真实保存
- 所有数据都是示例数据
