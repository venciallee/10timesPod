## 📊 技术规格

### 代码统计
- **总行数**: 656 行 HTML
- **CSS**: 使用 Tailwind CSS CDN（零配置）
- **JavaScript**: ~100 行原生 JS
- **依赖**: 仅 Tailwind CSS CDN

### 浏览器兼容性
- ✅ Chrome/Edge（Chromium）
- ✅ Safari
- ✅ Firefox
- ⚠️ IE 不支持

### 性能特性
- 单页应用，无页面刷新
- 内联样式，首屏加载快
- CDN 加速（Tailwind CSS）
- 轻量级（< 100KB）

## 📝 使用说明

### 如何测试各个功能

1. **页面导航**
   - 点击顶部"发现"、"订阅"、"设置"切换页面
   - 点击播客卡片进入播放页

2. **字幕显示模式**
   - 进入播放页后，点击"仅英文"、"仅中文"、"双语"按钮
   - 观察字幕面板内容变化

3. **字幕跳转**
   - 点击任意字幕句子
   - 观察该句子高亮效果变化

4. **播放控制**
   - 点击大播放按钮
   - 观察图标从播放切换到暂停

5. **设置面板**
   - 进入设置页
   - 切换提供方下拉框
   - 尝试输入 API Key
